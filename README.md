Customer Shopping Trends Analysis

Project Overview
This project analyzes customer shopping behavior using SQL. The data was imported from Excel into a SQL database.

Project Goals
- Analyze purchasing patterns
- Identify trends by gender, category, and location
- Understand relationship between previous purchases and spending

Tools Used
- SQL
- Excel

Analysis
- Average purchase amount
- Gender based spending comparison
- Category analysis
- Previous purchases correlation

Key Insights
- Certain categories dominate purchases
- Spending patterns differ by gender and location
- More previous purchases tend to correlate with higher spending
